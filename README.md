## 기술 스택

- Front-end: Vue.cli, node.js, Vuetify, Vuex, bootstrap
- Back-end: Python,Spring
- Env & etc: AWS-EC2, labssafy, Jira, Jenkins
- API를 이용하여 서버 구동
- Oauth2.0을 이용한 소셜 로그인
- Vue.cli와 router를 이용하여 SPA를 구현함으로써 빠른 UI/UX 향상
